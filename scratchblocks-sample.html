This page demonstrates the scratchblocks library, which renders Scratch-style code blocks in HTML.

Block Example:

when green flag clicked
say [Hello, World!] for (2) secs
repeat ((10) + (count) :: operators)
  move (10) steps
  turn cw (15) degrees
  move (10) steps
  turn ccw (15) degrees
end

Inline Example:

You can also use inline blocks like move (10) steps within regular text!
