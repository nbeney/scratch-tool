Notes

  1. a b c d e f g h i j A few blocks use primitives (a list with a specific format) instead of opcodes in the Scratch File Format. Trying to add these blocks using their opcodes will crash the editor. A table of all primitives can be found here.
  2. Any nonexistent opcode will work, but "undefined" is the opcode that is assigned when a custom block is named a control character.
